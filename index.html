Le fichier de style « style.css » est manquant. Il est important pour le fonctionnement de Javascript hqx. Assurez-vous qu'il soit présent dans le même dossier que le fichier « index.html » et réessayez. Dans le cas échant, re-téléchargez cette application.
Javascript est désactivé ! Ce programme fonctionnant grâce à Javascript, vous ne pourrez pas l'utiliser. Cliquez ici pour savoir comment activer javascript.
Javascript hqx est incompatible avec votre navigateur car celui-ci ne supporte pas l'API Canvas. Mettez-le à jour, puis réessayez.

Ce programme vous permet d'upscaler des images, c'est-à-dire de les agrandir en augmentant la qualité (de préférence des sprites de jeux vidéos) directement sur votre navigateur web. Choississez l'image sur l'ordinateur OU entrez le chemin absolu de l'image ET choisissez ensuite l'algorithme d'agrandissement (plus le chiffre après hq est grand, plus la qualité sera meilleure mais le traitement plus long). Vous pouvez aussi utiliser une adresse URL vers une image sur un site web, mais il se peut que le site en interdise l'accès aux scripts.

Programme basé sur js-hqx. Licence ici. Lisez le README.txt pour plus d'informations sur la compatibilité et autres. Programme créé par Eliastik. Découvrez Data URI Encoder, un autre de mes programmes permettant de convertir des images et des textes sous la forme d'une URI encodée en Base64 directement sur votre navigateur web préféré.

Mode débogage activé.
L'image est sauvegardable sous le type PNG, en cliquant droit dessus et en cliquant ensuite sur « Enregistrer l'image sous… »
